ARCHITECTURE
===============================================================================

**Privacy Layer**: Differential Privacy (ε=1.0) → K-Anonymity (k=3) → AES-256-CBC
**Blockchain Layer**: Ethereum smart contracts for immutable data storage
**ML Layer**: Isolation Forest with automatic training (≥10 data points)
**Presentation Layer**: Streamlit dashboard with 5-page interface
